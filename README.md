# NotifyMe

[![Build Status](https://img.shields.io/travis/notifymehq/notifyme.svg?style=flat-square)](https://travis-ci.org/notifymehq/notifyme)

Common interface for notification services.

Supported Gateways:
* Slack
* HipChat
* Twilio
* Campfire
* Gitter
* PagerDuty
* Webhook
* Pushover
* Intercom (soon)
* Amazon SES (soon)
* Amazon SNS (soon)
* Twitter (soon)
* Mail (soon)
* Yo (soon)
